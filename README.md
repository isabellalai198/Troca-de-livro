📚 TrocaLivros

Um aplicativo mobile para troca de livros entre leitores, construído com React Native (Expo), React Navigation e AsyncStorage.
O objetivo é criar uma comunidade de leitores que compartilham seus livros de forma simples e divertida.

🚀 Funcionalidades

Explorar  
Feed de livros da comunidade com busca por título/autor e filtro por gênero (12 categorias).

Meus Livros  
Gerencie seus livros cadastrados. Segure um livro para editar ou remover. Botão "+" para adicionar novos.

Trocas  
Histórico de solicitações enviadas e recebidas com filtros. Aceite ou recuse propostas recebidas.

Perfil  
Dados pessoais editáveis, estatísticas de livros cadastrados, trocas feitas e pendentes.

🛠️ CRUD Completo

Adicionar livros com título, autor, gênero, estado de conservação, descrição e cor de capa personalizável.

Editar ou remover livros já cadastrados.

Solicitar trocas ao ver um livro da comunidade, escolhendo qual dos seus oferecer.

Aceitar ou recusar solicitações recebidas.

Tudo salvo localmente com AsyncStorage.

✨ Extras de UX

6 livros de exemplo já populados para demonstração.

Animações de pressão nos cards.

Feedback tátil (haptics) nas ações principais.

Estados vazios informativos em todas as listas.

Badge de notificação na aba Trocas quando há propostas pendentes.

Tema claro/escuro automático com paleta terracota/âmbar.

🧩 Tecnologias utilizadas

React Native (Expo)

React Navigation (Stack e Tab)

AsyncStorage

CRUD completo com interface intuitiva e foco em UX.
